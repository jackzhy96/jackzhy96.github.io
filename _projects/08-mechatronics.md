## Overview

Course project at UC Berkeley involving the complete design and manufacturing process of a mechatronic system for object tracking.

## Technical Details

- Designed mechanical components using CAD software
- Fabricated parts using machine shop equipment and 3D printing
- Integrated sensors and actuators for closed-loop control
- Implemented tracking algorithms for real-time object following
