## Overview

Course project at UC Berkeley involving the complete design and manufacturing process of a mechatronic system for object tracking.

<figure>
  <img src="/images/projects/tri_bot_demo.gif" alt="Description" style="width: 85%;">
  <figcaption>Object Tracking Demo, a red dot is shown on the cell phone as the tracking target.</figcaption>
</figure>

## Technical Details

- Designed mechanical components using CAD software
- Manufactured parts using machine shop and 3D printing
- Integrated sensors and actuators for closed-loop control
- Implemented tracking algorithms for real-time object tracking using LabVIEW
